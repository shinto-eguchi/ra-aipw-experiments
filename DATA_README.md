# Data

This repository does not distribute individual-level NHEFS data.

For the NHEFS-based semi-synthetic experiment, place a prepared covariate CSV file at

```text
nhefs_covariates.csv
```

or set the environment variable

```text
NHEFS_CSV=/path/to/nhefs_covariates.csv
```

Then run

```r
source("00_prepare_nhefs_covariates.R")
```

which creates

```text
nhefs_covariates_prepared.csv
```

The prepared file is used by

```r
source("03_nhefs_semisynthetic_experiment.R")
```

Only covariates should be included in the input file. The response and outcome variables used in the semi-synthetic experiment are generated by the script.
